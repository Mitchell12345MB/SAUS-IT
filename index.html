About SauS-IT

I'm a hobbyist who runs an IT services company. I've been in the IT field for over 10 years now.

I started out making small minecraft plugins for my old minecraft server, learning Java and Maven along the way.

From there I started taking classes to not only learn how to make small plugins, but to make websites like this.

Eventually I started learning how to make video games using Unreal Engine 5. I'm currently learning how to make a open world RPG in Unreal Engine 5.

My projects range from small Minecraft plugins to some Windows OS modifications, Learning the ins and outs of Unreal Engine 5, and even some website development.
